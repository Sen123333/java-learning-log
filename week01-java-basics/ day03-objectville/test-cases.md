# Day 3 Test Cases

## 1. DogTestDrive 测试用例

**测试目标**：Dog 类（name, size, breed 字段 + bark() 方法）

- **正常情况**：
  - 创建 Dog 对象，name="小黑", size=40, breed="拉布拉多" → 调用 bark() 输出正确
  - 创建 3 只不同属性的狗 → 每只狗都能正常叫

- **边界情况**：
  - size = 0（最小体型）
  - size = 100（很大体型）
  - name 为空字符串

## 2. StudentTestDrive 测试用例

**测试目标**：Student 类（姓名、学号、成绩等字段 + 方法）

- **正常情况**：
  - 创建学生对象，设置成绩 → 调用显示信息方法
  - 计算平均分或等级

- **边界情况**：
  - 成绩 = 0、60、100
  - 姓名为空

- **无效情况**：
  - 成绩 < 0 或 > 100

## 3. BookTestDrive 测试用例

**测试目标**：Book 类（书名、作者、页数等字段 + 方法）

- **正常情况**：
  - 创建图书对象，调用 displayInfo() 或 borrow() 方法
- **边界情况**：
  - 页数 = 1（最小书）
  - 页数 = 1000（大部头）
- **无效情况**：
  - 书名为空

## 4. SimpleGuessingGame 测试用例

**测试目标**：猜数字游戏（Number 类）

- **正常情况**：
  - 输入正确答案（如 7）→ 输出 "Correct!"
  - 输入小于答案 → "Too low"
  - 输入大于答案 → "Too high"

- **边界情况**：
  - 输入 0、Integer.MAX_VALUE

- **无效情况**：
  - 输入非整数（需异常处理）
