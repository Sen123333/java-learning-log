## 《Head First Java》第2章 5个核心概念总结

1. **对象（Objects）**：Java程序的核心是对象，对象有状态（实例变量）和行为（方法）。

2. **类（Class）**：类是对象的蓝图，定义了对象的结构和行为。每个对象都是某个类的实例。

3. **实例变量（Instance Variables）**：每个对象都有自己独立的实例变量，用于保存对象的状态。

4. **方法（Methods）**：方法定义对象的行为，可以访问实例变量并执行操作。

5. **面向对象 vs 面向过程**：Java强调使用对象和类来组织代码，而不是单纯的过程式编程，这让代码更易维护和扩展。
