# Day 2 - 变量、If/Else、循环、方法

## 读书内容
- 《Head First Java》第二版 第2章（A Trip to Objectville）
- 重点学习面向对象基础：对象、类、实例变量、方法等概念

## 程序功能说明
1. OddEvenChecker.java —— 判断整数奇偶性
2. GradeLevel.java —— 根据分数输出等级（A/B/C/D/F）
3. TemperatureConverter.java —— 摄氏度与华氏度转换
4. MaxOfThree.java —— 求三个整数的最大值
5. MethodPractice.java —— 练习编写 isEven、maxOfThree、factorial 方法

## 运行方法
```bash
cd day02-variables-if-loop-method/src

javac *.java

java OddEvenChecker
java GradeLevel
java TemperatureConverter
java MaxOfThree
java MethodPractice
